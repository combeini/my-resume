## Markdown 简历渲染器
#### 简单实用说明
1. 下载index.html, 使用浏览器打开
2. 点击 `获取MD模板` 按钮,修改模板内markdown部分信息
3. 点击 `加载MD简历` 按钮,调整字体和主题
4. 右键页面,选择打印

#### 备注
由AI实现,本地渲染,无数据交互和存储,请妥善保管数据,无售后.
